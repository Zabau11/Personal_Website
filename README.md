# David Sabau - Personal Website

Personal portfolio and writing site for [davidsabau](https://zabau.ceo), built as a lightweight static website.

## Live Site

- [https://zabau.ceo](https://zabau.ceo)

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Project Structure

- `index.html` - main homepage
- `writing.html` - writing archive index
- `2025.html`, `2026.html` - yearly writing entries
- `styles.css` - global styles
- `script.js` - client-side interactions
- `images/` - site images
- `favicon.svg`, `background-vibe.svg` - visual assets

## Run Locally

Option 1 (quick preview):

1. Open `index.html` directly in your browser.

Option 2 (local server):

1. Run `python3 -m http.server 5500`
2. Visit `http://127.0.0.1:5500`

## Notes

- The site includes Vercel Web Analytics (`/_vercel/insights/script.js`).
